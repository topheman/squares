Qu'est ce que Topheman Squares ?
Une adaptation - d'après les souvenirs que j'en ai - en HTML5/Javascript d'un jeu auquel je jouais dans les années 1990 sur notre IIvi (notre premier Mac couleur).
Comment jouer à Topheman Squares ?
  • Pour passer au niveau supérieur, vous devez soit avoir recouvert 80% de la zone de jeu, soit avoir tué tous les ennemis.
  • Pour pouvoir capturer un ennemi, vous devez attraper un des points oranges qui apparaissent périodiquement.
  • Plus vous faites de grandes formes, plus vous gagnez de points.
Pourquoi choisir HTML5/Javascript comme technos ?
J'aime beaucoup coder en Javascript et c'était l'occasion d'utiliser la feature canvas proposée par html5.
Requis
Tout ce dont vous avez besoin pour jouer est un navigateur récent.
Voici la liste des OS / navigateurs sur lesquels j'ai testé mon jeu (et qui marchent) :
  • Windows : Chrome 16, Firefox 3.6.25, Safari 5.1.2
  • Mac : Firefox
  • Linux :
    • Ubuntu 11.04 : Chrome 14, Rekonq 0.7.0, FireFox 4
Je compte sur vous pour me remonter les éventuels bugs ou incohérences, n'hésitez pas à me contacter par mail ou à laisser un commentaire sur l'article que j'ai fait sur mon blog.

JOUER A TOPHEMAN SQUARES !

What is Topheman Squares ?
An adaptation in HTML5/Javascript - based on the memories I have, of a game I used to play in the 90s on our IIvi (our first color computer).
How to play Topheman Squares ?
  • To get to the next level, you have to cover at least 80% of the game zone or kill every enemies.
  • To kill an enemy, you have to get an orange dot that appears from time to time.
  • The bigger shapes you draw, the higher score you get.
Why HTML5/Javascript ?
I really enjoy coding in Javascript and that was the occasion of using the canvas feature of html5.
Requirements
All you need is a recent browser. Here's a list of the OS / browsers I've tested the game on :
  • Windows : Chrome 16, Firefox 3.6.25, Safari 5.1.2
  • Mac : Firefox
  • Linux :
    • Ubuntu 11.04 : Chrome 14, Rekonq 0.7.0, FireFox 4
Report any bug or inconsistencies to help improve the game. You can mail me or leave a comment on this blog post.
